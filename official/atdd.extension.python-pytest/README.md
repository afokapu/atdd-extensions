# atdd.extension.python-pytest

Official ATDD Python pytest extension.

This extension establishes the Python pytest validator workspace contract.

It is the first extraction target for Python-specific ATDD validator behavior currently entangled with ATDD core.

## Purpose

This extension defines:

- Python runtime contract
- pytest runner contract
- Python validator layout
- Python fixture conventions
- ATDD violation output expectations
- legacy validator compatibility path
